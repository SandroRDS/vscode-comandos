VSCode - Comandos Úteis

Geral

Abrir palheta de comandos CTRL + SHIFT + P ou F1
Pesquisar por arquivos/diretórios do projeto por nome CTRL + P
Abrir configurações CTRL + ,

Navegação

Ir para a linha... CTRL + G
Buscar por seletor... CTRL + SHIFT + O
Navegar pelos arquivos abertos CTRL + SHIFT + TAB

Edição Básica

Recortar linha inteira (seleção vazia) CTRL + X
Copiar linha inteira (seleção vazia) CTRL + C
Mover linha para cima / baixo ALT + ⬆ / ⬇
Copiar linha para cima / baixo SHIFT + ALT + ⬆ / ⬇
Excluir linha CTRL + SHIFT + K
Inserir linha acima CTRL + SHIFT + ENTER
Indentar para frente / trás CTRL + ] / [
Mover cursor para o início da linha HOME
Mover cursor para o fim da linha END
Mover cursor para o início do arquivo CTRL + HOME
Mover cursor para o fim do arquivo CTRL + END
Rolar página para cima / baixo CTRL + ⬆ / ⬇
Dobre / Desdobre tag CTRL + SHIFT + [ / ]
Adicionar / Remover comentário de linha CTRL + /
Ativar / Desativar quebra de linha automática ALT + Z

Pesquise e Substitua

Encontrar CTRL + F
Subtituir CTRL + H
Navegar para o próximo / anterior F3 / SHIFT + F3
Selecionar todas as ocorrências do Encontrar ALT + ENTER
Adicionar seleção à próxima ocorrência do Encontrar CTRL + D

Multi-cursor e Seleção

Inserir cursor com mouse ALT + ტ
Inserir cursor acima / abaixo CTRL + ALT + PgUp / PgDn
Selecionar linha inteira atual CTRL + L
Selecionar todas as ocorrências da seleção atual CTRL + SHIFT + L
Selecionar todas as ocorrências da palavra atual CTRL + F2

Exibição

Alternar visibilidade da barra lateral CTRL + B
Abrir pré-visualização de Markdown CTRL + SHIFT +
Abrir pré-visualização de Markdown ao lado CTRL + K depois V
Modo Zen (ESC ESC para sair) CTRL + K depois Z

Depurar

Adicionar / Remover breakpoint F9
Iniciar / Continuar F5
Parar SHIFT + F5
Passo para dentro / fora F11 / SHIFT + F11
Passar por cima F10

Terminal Integrado

Abrir / Fechar terminal integrado CTRL + '
Criar novo terminal integrado CTRL + SHIFT + '
Abrir / Fechar console de depuração CTRL + SHIFT + Y
Abrir / Fechar Saída CTRL + K depois CTRL + H
Abrir / Fechar Problemas CTRL + SHIFT + M

Gerenciamento de Arquivos

Mostrar na pasta do Explorer o arquivo ativo CTRL + K depois R
